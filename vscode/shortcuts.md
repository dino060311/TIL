# 🚀 VSCode 단축키 (Windows 기준)

매일 코딩하면서 손가락이 먼저 반응해야 하는 핵심 단축키.

## 1. 코드 편집 & 재배치

|         단축키         |       기능       | 활용 팁                                                |
| :--------------------: | :--------------: | :----------------------------------------------------- |
|   `Alt` + `↑` / `↓`    | **줄 위치 이동** | 코드 순서 바꿀 때 씀. 잘라내기 안 해도 돼서 편함.      |
| `Shift` + `Alt` + `↓`  |   **줄 복사**    | 아래로 바로 복제함. 변수 선언이나 로그 찍을 때 필수임. |
| `Ctrl` + `Shift` + `K` | **줄 전체 삭제** | 백스페이스 연타할 필요 없음. 깔끔하게 한 줄 지움.      |
|    `Ctrl` + `Enter`    | **다음 줄 시작** | 커서가 중간에 있어도 바로 아래 새 줄로 넘어감.         |
|      `Ctrl` + `/`      |  **주석 토글**   | 한 줄 혹은 블록 전체 주석 처리할 때 씀.                |

## 2. 멀티 커서 & 일괄 수정

|         단축키         |          기능          | 활용 팁                                             |
| :--------------------: | :--------------------: | :-------------------------------------------------- |
|      `Ctrl` + `D`      |   **단어 다중 선택**   | 같은 단어 하나씩 추가 선택함. 몇 개만 바꿀 때 좋음. |
| `Ctrl` + `Shift` + `L` |   **단어 전체 선택**   | 파일 내 동일 단어 모두 선택함. 일괄 수정 시 최고임. |
|          `F2`          | **이름 변경 (Rename)** | 변수명 바꿀 때 씀. 참조된 곳까지 안전하게 다 바뀜.  |
|      `Alt` + 클릭      |   **커서 다중 생성**   | 원하는 곳마다 클릭해서 동시에 타이핑함.             |

## 3. 탐색 & 뷰 (흐름 끊기 방지)

|         단축키         |         기능         | 활용 팁                                               |
| :--------------------: | :------------------: | :---------------------------------------------------- |
|      `Ctrl` + `P`      |  **파일 빠른 열기**  | 파일 이름만 쳐서 바로 이동함. 탐색기 안 뒤져도 됨.    |
|         `F12`          |   **정의로 이동**    | 메서드가 선언된 곳으로 점프함. 로직 파악할 때 필수임. |
|      `Alt` + `←`       | **이전 위치로 이동** | `F12` 타고 갔다가 다시 원래 코딩하던 곳으로 돌아옴.   |
| `Ctrl` + `Shift` + `\` |   **괄호 짝 이동**   | `{ }` 사이가 멀 때 짝이 맞는지 바로 확인 가능함.      |
|      `Ctrl` + `B`      |  **사이드바 토글**   | 코드 화면 넓게 쓰고 싶을 때 탐색기 창 숨김.           |

# Chapter 03. 반복문과 배열 그리고 예외 처리

## 📅 2026-04-19 (for 문을 활용한 반복 합계 계산)

### ✅ 배운 내용

**for 문의 특징**

- 주로 반복 횟수가 명확히 정해져 있는 경우 사용함.
- 초기화식, 조건식, 반복 후 작업이 한 줄에 명시되어 있어 가독성이 좋습니다.

**초기문**  
for 문에서 초기문은 주로 조건식에서 사용하는 변수를 초기화함.

- 초기문은 시작할 때 한 번만 수행함.
- 콤마(,)로 분리하여 여러 문장을 나열할 수 있음.
- 초기문은 빈 상태로 두어도 되지만 끝에 세미콜론(;)은 있어야 함.

**조건식**  
매 루프 시작 전 검사하며, 조건식의 결과가 true이면 반복이 계속되며 false이면 for 문을 벗어남

```bash
for(초기문; true; 반복 후 작업) { //무한 반복
    ................
}
for(초기문; ; 반복 후 작업) { // 조건식이 비어 있으면 무한 반복
    ................
}
```

**반복 후 작업**

- 루프 몸체 실행 후 실행됨. 주로 루프 변수를 증감시키는 용도로 사용함.
- 반복 후 작업문에는 콤마(,)로 분리하여 여러 문장을 나열할 수 있음.

```bash
for(i = 0;, i < 10; i++, System.out.println(i)){
    ................
}
```

**반복문에 변수 선언**

- `for` 문의 초기화식에서 선언된 변수 i는 해당 for 문 블록 내부에서만 유효한 지역 변수임.
- 루프가 종료되면 해당 변수는 메모리에서 사라지므로 외부에서 접근할 수 없음.

```bash
for(int i = 0; i < 10; i++)
    System.out.print(i);
```

### ⚠️ Troubleshooting

- **문제**: `Syntax error on token ",", delete this token` 메시지 발생.
- **원인**: `for` 문의 소괄호 () 안에서 구문을 구분할 때 세미콜론(;) 대신 쉼표(,)를 사용함.
- **해결**: 문장의 끝을 의미하는 세미콜론을 사용하여 각 식을 명확히 구분함.

## 📅 2026-04-20 (while 문)

### ✅ 배운 내용

**while 문의 특징**

- **반복 횟수가 불분명할 때**: 사용자의 입력이나 특정 상태 변화에 따라 루프를 종료해야 하는 경우에 매우 적합합니다.
- **조건식 필수**: `for` 문과 달리, `while()` 소괄호 안에 조건식이 없으면 컴파일 오류가 발생함.
- **종료 조건의 중요성**: 루프 몸체 안에서 조건식의 결과가 언젠가 `false`가 되도록 설계해야 합니다. 그렇지 않으면 메모리 자원을 계속 점유하는 무한 루프에 빠지게 됩니다.

**데이터 타입 변환**

- 정수 데이터들의 평균을 구할 때 소수점까지 정확히 표현하기 위해 `(double)` 강제 형변환을 적용함.

## 📅 2026-04-21 (do-while 문)

### ✅ 배운 내용
**1. do-while 문의 특징**

- 선 실행, 후 검사: 조건식을 검사하기 전에 루프 몸체(do 블록)를 최소 한 번은 반드시 실행함.
- 세미콜론(;) 주의: `while (조건식);` 처럼 문장 끝에 반드시 세미콜론을 붙여야 컴파일 오류가 나지 않음.

**2. 문자의 산술 연산과 형변환**

- 자바에서 char 타입에 숫자를 더하면 결과가 int 타입으로 변함.
- 따라서 다시 문자로 출력하려면 (char)를 이용해 강제 형변환을 해줘야 함.

**3. while 문과의 차이점**
| 구문 | 조건 검사 시점 | 최소 실행 횟수 |
|:---:|:---:|:---:|
| while | 작업문 진입 전 (선 검사) | 0회 |
| do-while | 작업문 실행 후 (후 검사) | 1회 |

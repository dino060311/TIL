# 산술 연산자를 활용한 시간 계산 및 나머지 연산 활용

### ✅ 오늘 배운 내용
- 산술 연산자는 `+`(더하기), `*`(곱하기), `/`(나누기), `%`(나머지)
- `%` 연산자
  정수 n이 홀수인지 짝수인지 구분할 때 유용하게 사용함.
  `int r = n % 2; //n이 홀수이면 r은 1, 짝수이면 r은 0`
  다음과 같이 n 값이 5의 배수인지 확인하기 위해서도 사용됨.
  `int s = n % 5; //n이 5의 배수이면 s는 0`

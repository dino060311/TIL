# git clone 사용법

집 컴퓨터에서 작업하던 내용을 노트북에서 이어하고 싶을 때, 혹은 그 반대의 상황에서 사용하는 명령어

## 새로운 기기에서 시작할 때 (`clone`)

노트북에 프로젝트 폴더가 아예 없을 때, 깃허브에 있는 저장소를 통째로 복사해옵니다.

1. 깃허브에서 주소 복사하기  
   본인의 깃허브 저장소로 접속합니다.  
   초록색 [<> Code] 버튼을 누릅니다.  
   HTTPS 탭 아래에 있는 주소(예: https://github.com/아이디/폴더명.git)를 복사합니다.

2. 터미널(또는 CMD) 열기 
   저장할 위치로 이동한 뒤 아래 명령어를 입력합니다.  
   ```bash
   git clone https://github.com/아이디/폴더명.git
   ```

## 데이터 충돌 방지를 위한 필수 Git 명령어

노트북 작업 완료 시:
```bash
git add .
git commit -m " "
git push origin main
```


다시 집 컴퓨터로 돌아갔을 때:  

작업을 시작하기 전 무조건 `git pull origin main`  
이렇게 하면 집과 노트북 사이에서 데이터가 꼬이지 않고 안전하게 이어갈 수 있습니다.

# git clone 사용법

집 컴퓨터에서 작업하던 내용을 노트북에서 이어하고 싶을 때, 혹은 그 반대의 상황에서 사용하는 명령어

## 새로운 기기에서 시작할 때 (`clone`)

노트북에 프로젝트 폴더가 아예 없을 때, 깃허브에 있는 저장소를 통째로 복사.

### 깃허브에서 주소 복사하기
   1. 본인의 깃허브 저장소로 접속
   2. 초록색 [<> Code] 버튼 클릭
   3. HTTPS 탭 아래에 있는 주소를 복사

### Git Bash 실행
   1. 저장할 위치로 이동한 뒤 폴더 빈 공간에서 `Shift + 마우스 우클릭`  
   2. 메뉴에서 **[Open Git Bash here]**를 클릭
   3. 터미널 창에 아래 명령어를 입력
   ```bash
   git clone https://github.com/아이디/폴더명.git
   ```

## 데이터 충돌 방지를 위한 필수 Git 명령어

노트북 작업 완료 시:
```bash
git add .
git commit -m " "
git push origin main
```


다시 집 컴퓨터로 돌아갔을 때:  

작업을 시작하기 전 무조건 `git pull origin main`  
이렇게 하면 집과 노트북 사이에서 데이터가 꼬이지 않고 안전하게 이어갈 수 있습니다.
